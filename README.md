# XComiK

A simple iOS App to view [xkcd](http://www.xkcd.com) comics.

This is my first iOS app.

### Current features
- View current comic
- View the latest 100 comics

## Usage

Clone this repository and execute
```
pod install
```

Open the xcworkspace in Xcode
